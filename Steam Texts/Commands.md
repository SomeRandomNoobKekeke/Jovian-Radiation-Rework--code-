[h2] Console commands [/h2]

[table]
    [tr]
        [td]rad[/td]
        [td]
        Command for editing current config
        Without arguments just prints the whole config
        [/td]
    [/tr]
    [tr]

        [td]prints the value of the setting[/td]
    [/tr]
    [tr]
        [td]rad [setting path] [value][/td]
        [td]sets some setting[/td]
    [/tr]
    [tr]
        [td]rad_save[/td]
        [td]Saves current config somewhere[/td]
    [/tr]
    [tr]
        [td]rad_load[/td]
        [td]loads config from somewhere[/td]
    [/tr]
    [tr]
        [td]rad_amount [amount][/td]
        [td]prints / sets radiation front x coordinate[/td]
    [/tr]
    [tr]
        [td]rad_vanilla[/td]
        [td]Another way to access vanilla radiation params, here you can also reset them to defaults[/td]
    [/tr]
    [tr]
        [td]rad_debugmodel[/td]
        [td]toggles debug of some model[/td]
    [/tr]
    [tr]
        [td]rad_server_debugmodel[/td]
        [td]toggles debug of some model on server[/td]
    [/tr]
    [tr]
        [td]rad_printmodel[/td]
        [td]Prints current model[/td]
    [/tr]
